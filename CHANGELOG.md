# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-10

### Added

- Remaining-quota overlay for Codex's 5-hour and 7-day rolling windows.
- Reset countdown display with hour and day-aware formatting.
- One-second hover state machine with immediate hide and repeatable entry.
- Automatic pet-position tracking from Codex local state.
- Official local app-server usage source with structured-state, JSONL, and cache fallbacks.
- Click-through overlay, menu-bar controls, detection-area diagnostics, and launch-at-login support.
- PyInstaller packaging and macOS permission diagnostics.
